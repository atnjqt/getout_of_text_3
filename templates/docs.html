App Documentation & Route Reference

This page documents the main routes/endpoints of the getout-of-text3 SCOTUS dashboard and their functions.

Route Method Description
/ GET Main search page for SCOTUS term database. Search, filter, and view case data.
/data.json GET Returns all case data as JSON for use in the frontend or API clients.
/data.html GET Returns all case data as an HTML table (for direct viewing or embedding).
/search GET Advanced search endpoint. Returns filtered case data with context snippets and highlighting.
/oyez GET Oyez JSON viewer: input an Oyez API URL or use auto-fill from search results to visualize case metadata, justices, and opinions.
/about GET About page: project proposal, developer info, references, and dark mode toggle.

For more details, see the About page or the GitHub repository.
